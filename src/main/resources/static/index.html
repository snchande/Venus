Setup Requirements

C# and F# cells require the .NET SDK — no extra tools needed:

dotnet --version   # verify SDK is installed
dotnet fsi --version  # verify F# Interactive

Install the .NET SDK from https://dot.net (free), then restart Venus.

C++ Built-in Libraries

C++ cells run with a pre-compiled preamble. The headers below are available in every cell — no installation needed. Third-party package management (vcpkg, Conan) is not yet supported.

Standard Library Headers — always available
📦 I/O & Streams <iostream> <sstream> <fstream> <iomanip> 📦 Containers <vector> <map> <unordered_map> <set> <list> <deque> <queue> <stack> <array> 📦 Algorithms & Math <algorithm> <numeric> <functional> <cmath> <random> 📦 Modern C++ <optional> <variant> <tuple> <memory> <string> <string_view> 📦 Utilities <chrono> <typeinfo> <stdexcept>

Also active: using namespace std; and _USE_MATH_DEFINES (for M_PI, M_E etc. on MSVC)

Venus Display Helpers
venusHtml(string)Renders HTML output inline in the cell
venusDisplay(value)Prints any streamable value
venusTable(vector<T>)Shows item count summary
venusTable(map<K,V>)Renders a key/value ASCII table
Using Additional Headers

Any system header not listed above can be included at the top of your cell:

#include <regex>
#include <filesystem>   // C++17 — may need -lstdc++fs on older g++
#include <thread>

// Your cell code follows…

⚠ Third-party headers (Boost, Eigen, nlohmann/json, etc.) require manual compiler setup — install the library system-wide so the compiler can find them. Venus does not manage C++ package installation.
